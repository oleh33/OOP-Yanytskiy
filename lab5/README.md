# Lab5 v20 — Генерики, колекції, LINQ, винятки

## Тема
Простий аналітик даних (DataFrame / Row) з узагальненими агрегаторами та обробкою винятків.

## Мета
Навчитися працювати з узагальненими типами (Generics), колекціями та LINQ, а також коректно валідувати дані й використовувати власні винятки.

## Реалізовано
- `DataFrame` з композицією `DataRow` (кожен рядок містить словник колонок).
- Запити LINQ: `Where`, `Select`, `GroupBy`.
- Узагальнений клас `Aggregator<T>` із методами `Sum`, `Avg`, `Count`.
- Власний виняток `ColumnNotFoundException`, який кидається при зверненні до неіснуючої колонки.
- Валідація даних (порожня назва колонки, null-рядок).
- Обробка винятків у `Program.cs` за допомогою `try-catch`.

## Приклад запуску
Програма створює 10 продуктів з полями:
- Name
- Price
- Quantity
- Category

Далі виконується:
- Фільтрація продуктів з ціною > 20.
- Обчислення суми та середнього значення ціни.
- Групування по Category (Food / Tech).
- Демонстрація винятку при зверненні до неіснуючої колонки.

## Команда запуску
```bash
dotnet run
```
