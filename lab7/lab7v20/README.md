# Лабораторна робота №7  
## Тема: Обробка IO/мережевих помилок та патерн Retry

### Мета
Навчитися обробляти IO/мережеві помилки та реалізувати патерн Retry з експоненційною затримкою.

### Короткий опис
Реалізовано `FileProcessor`, `NetworkClient`, узагальнений `RetryHelper` і сценарій відправки push-сповіщення, де перші спроби падають з помилками, а потім операція успішна.
